# SOEN357-Languify

Languify is an AI-powered language learning app designed to provide a personalized and interactive learning experience. The app adapts to each user’s proficiency level and goals, creating a custom learning path that enhances grammar, pronunciation, vocabulary, and listening skills.

Key Features:
Personalized Learning Path – Users take an initial assessment to tailor lessons based on their skill level, learning goals, and time commitment.
AI Tutoring System – A smart tutor provides real-time feedback on pronunciation, grammar, and vocabulary through interactive conversation simulations.
Interactive Learning Activities – Engaging exercises, including flashcard quizzes, grammar drills, role-playing, and gamified challenges, make learning fun and effective.
User-Friendly Structure – The app features a home dashboard, personalized learning sections, an AI tutor interface, practice sessions, and detailed progress analytics.
Languify ensures an intuitive and immersive learning journey, making language acquisition both efficient and enjoyable.
